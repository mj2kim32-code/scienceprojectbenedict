Instructions
 1. Measure a fixed volume of 50mL of sulfuric acid by using a measuring cylinder and pour it into a scientific thermos.
2. Put in a temperature probe and start measuring in the Vernier app.
3. Add a fixed length of magnesium ribbon (15cm) when the temperature settles.
4. Record the data for 40 seconds and then stop the recording.
5. Calculate temperature change (final temperature − initial temperature).
6. Use temperature change to calculate the increase rate (temp. change/time).
7. Repeat the experiment for different concentrations (0.5 M → 1.5 M).
8. Repeat each trial and calculate the average for accuracy.
